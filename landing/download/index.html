Frequently Asked Questions

Can I use my own GPU server instead of the public one? ▼
Yes! In VS Code settings, change vscodeOllama.host to your own Ollama server address. You can run Ollama on any GPU hardware (NVIDIA, AMD, or even macOS with Metal acceleration).
Do I need the Chrome extension to use completions? ▼
No. The extension is optional and only adds voice input. You can use code completions without it. The extension is recommended for hands-free development.
What are the system requirements? ▼
Minimum: 2GB RAM, 1GB disk space. Recommended: 4GB+ RAM, SSD. Works on Intel and Apple Silicon Macs, modern Linux (Ubuntu 20.04+), Windows 10/11.
Is my code private? ▼
Yes. Completions are processed on the GPU server, but you control where that server is. You can run everything on your local machine. No telemetry or code collection.
How do I update to a new version? ▼
Download the new version and run the installer over your existing installation. Your settings are preserved. Or enable auto-updates in settings.
Can I use WPBasic for non-WordPress projects? ▼
Yes! WPBasic is just a variant with WordPress patterns pre-loaded. You can use it for any PHP project. It includes the same features as Standard, just with WordPress-focused QA checks.
What's the difference between Standard and WPBasic? ▼
Standard: General-purpose AI completions for all languages. WPBasic: Same features, but with WordPress-specific QA checks, theme/plugin patterns, and PHP-focused completions.
Is there a portable/USB version? ▼
Portable versions are planned for v1.1. Currently, you need to install the application. On macOS/Linux, you can copy the application folder anywhere.
