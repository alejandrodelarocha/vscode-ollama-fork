# VS Code Ollama Fork - Logo Design

## Design Concept

The logo combines code brackets with neural network connectivity, representing the intersection of code editing and AI-powered intelligence.

### Visual Elements

**Code Brackets `< >`**
- Left bracket in blue gradient (cyan → blue)
- Right bracket in purple gradient (purple → dark purple)
- Represent the core function: code completion within an editor
- Angular, modern aesthetic matching VS Code's design language

**Neural Network Core**
- Four nodes arranged in diamond pattern
- Connected by subtle lines showing information flow
- Represents the AI model processing code context
- Minimalist, abstract representation of machine learning

## Usage Guidelines

### Do ✅
- Use full-color versions on contrasting backgrounds
- Maintain minimum 32px size for icon usage
- Keep aspect ratio when resizing
- Use dark variant on light backgrounds
- Use light variant on dark backgrounds
- Apply consistent padding (1:1 padding-to-size ratio)

### Don't ❌
- Rotate or distort the logo
- Change colors without brand approval
- Use on backgrounds with insufficient contrast
- Embed text directly on logo
- Stretch disproportionately
- Outline or add shadows

## Design Rationale

### Why Code Brackets?
- Instantly recognizable as programming context
- VS Code's iconic bracket highlighting
- Universally understood symbol for code completion

### Why Neural Network?
- Represents the AI model (Ollama) processing code
- Subtle, abstract—not literal or clichéd
- Conveys intelligence without complexity
- Connects to the "learning" aspect of AI

### Why These Colors?
- **Blue**: Trust, VS Code brand, professional
- **Purple**: AI/ML, innovation, magic
- **Gradients**: Modern, depth, visual interest
- **High Contrast**: Accessible, readable at small sizes
